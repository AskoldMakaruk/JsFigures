About

Simple program in javascript with jquery, that takes user's input for 3 dots on canvas and draws parallelogram, which lies on them. After parallelogram has been built, program calculates area of it using Gauss's area formula and builds circle with this area and center in the parallelogram's center of mass. Dots can be rearranged by user.

Known bug: if you scroll down coordinate system no longer works with canvas properly.
